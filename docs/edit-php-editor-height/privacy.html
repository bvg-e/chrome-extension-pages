Privacy Policy

For the Chrome extension Modify PHP Editor Height.

Overview

Modify PHP Editor Height is a Chrome extension that allows the user to change the height of the PHP editor area on supported Bitrix pages.

Data Collection

This extension does not collect personal data, authentication data, browsing history, financial information, health information, messages, or any other sensitive user data.

Data Storage

The extension stores only one local setting: the user-selected editor height value. This setting is stored locally using Chrome extension storage and is used only to remember the preferred editor height on supported pages.

Data Sharing

The extension does not sell, transfer, or share user data with third parties.

Remote Code and External Requests

The extension does not load or execute remotely hosted code. The extension does not send stored settings or page content to external servers.

Purpose of Use

The stored setting is used exclusively to provide the extension’s single purpose: adjusting the visible height of the PHP editor area on supported pages.

Contact
