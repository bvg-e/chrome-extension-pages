Privacy Policy

For the Chrome extension Bitrix Task Chat Sorter.

Overview

Bitrix Task Chat Sorter is a Chrome extension that displays a separate list of Bitrix task chats sorted by the latest user activity without system messages. It is intended to help users find active task chats and open tasks faster inside the supported Bitrix portal.

Data Collection

This extension does not collect personal data, authentication data, financial information, health information, browsing history, or other sensitive user data for sale or transfer to third parties.

Data Storage

The extension stores only local extension data required for its operation, including cached chat list data and interface state such as panel position. This data is stored locally using Chrome extension storage and is used only to support the extension’s functionality on the supported Bitrix portal.

Data Sharing

The extension does not sell, share, or transfer stored extension data to third parties.

External Requests and Remote Code

The extension does not load or execute remotely hosted code. All executable code is included in the extension package. Network requests are used only to interact with the supported Bitrix portal in order to display and update task chat data.

Purpose of Use

Any locally stored extension data is used exclusively to provide the extension’s single purpose: displaying a sorted list of Bitrix task chats and related interface state on the supported portal.

Supported Website

This extension is intended to operate only on the supported Bitrix portal: b24.it-solution.ru.

Contact

If you have questions about this Privacy Policy, you can contact the developer at: bvg.extensions@gmail.com
